# diagrams/

Lưu sơ đồ đi kèm DD. Khuyến nghị tên file:

```text
context.mmd                  # C4 context module
container.mmd                # nếu cần
feature-flow-[Fxx].mmd       # flowchart feature
sequence-[Fxx].mmd           # sequence diagram feature/function
state-[entity].mmd           # state machine
ERD-[module].mmd             # entity relation
```

Các sơ đồ phải có mã liên kết đến Feature/Function/View hoặc Business Rule tương ứng. Sơ đồ không thay thế bảng luồng chi tiết trong các file Markdown.
