## 9. Ghi chú triển khai
- Tài liệu này mô tả **một chức năng riêng lẻ**
- Mọi bước đều phải đủ chi tiết để dev có thể coding trực tiếp
- Nếu có nhiều luồng con, tách riêng từng flow rõ ràng
- Nếu có realtime, phải nêu rõ sự kiện nào cập nhật UI
- Nếu có SQLite cache, phải mô tả rõ cơ chế đồng bộ với Supabase
