# Overview

Module này kết nối người dùng vào nhóm Zalo chăm sóc đặc biệt để tăng khả năng hỗ trợ, tương tác và chăm sóc ngoài ứng dụng.

Mục tiêu chính:
- Tạo kênh hỗ trợ riêng.
- Hỗ trợ chăm sóc sâu hơn.
- Tăng kết nối giữa hệ thống và khách hàng.

Phạm vi:
- Áp dụng cho các khách hàng thuộc nhóm chăm sóc đặc biệt.
