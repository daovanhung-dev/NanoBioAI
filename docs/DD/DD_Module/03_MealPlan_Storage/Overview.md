# Overview

Module này lưu lại các thực đơn đã được hệ thống sinh ra để người dùng có thể xem lại, đối chiếu và sử dụng ở các chu kỳ sau.

Mục tiêu chính:
- Bảo toàn dữ liệu thực đơn.
- Hỗ trợ tra cứu lịch sử.
- Tạo nền tảng cho việc so sánh và làm mới chu kỳ.

Phạm vi:
- Áp dụng cho mọi thực đơn đã được tạo trong hệ thống.
